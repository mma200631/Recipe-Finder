
# Savorly

## Overview

As a software developer, I am continually building projects that strengthen my understanding of JavaScript and modern web development. My goal with this project was to create a fully interactive web application that integrates external data, responds to user interactions, and stores user data locally in the browser. This project allowed me to gain practical experience with JavaScript modules, asynchronous programming, DOM manipulation, and browser storage.

Savorly is a recipe discovery and meal planning web application. Users can search for recipes, view detailed cooking instructions, save their favorite recipes, and create a weekly meal plan with breakfast, lunch, and dinner for each day. Recipe information is retrieved dynamically from TheMealDB API, while favorites and meal plans are stored using Local Storage to provide a personalized experience.

The purpose of writing this software was to improve my understanding of JavaScript by building a real-world application. Through this project, I learned how different JavaScript modules communicate with one another, how to consume REST APIs using `fetch()`, how to dynamically update the DOM, and how to persist data using Local Storage.
