# Overview

Hello World - JavaScript

As a software engineer, I am continually building my programming skills by learning new languages and technologies. This project is my first step in learning JavaScript and understanding how it interacts with HTML to create dynamic web pages.

This software is a simple "Hello, World!" web application that displays the message **"Hello, World!"** in a web browser using JavaScript.

The purpose of creating this software is to become familiar with the JavaScript language, understand how to link JavaScript to an HTML document, and practice manipulating webpage content through the Document Object Model (DOM).
